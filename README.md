PowerBI - First project 

Power BI Data Visualization Project: Retail & Sales Insights

📌 Project Overview
This project focuses on transforming raw business data into interactive, actionable insights using Power BI Desktop. Completed during the Data Technician bootcamp, the project covers the end-to-end business intelligence workflow: from connecting to diverse data sources (Excel and OData) to publishing live reports in the Power BI Service.

🛠️ Key Skills & Technical Features
Data Integration & Transformation
Multi-Source Connection: Integrated data from local Excel workbooks and external OData feeds to create a unified data model.
Power Query (ETL): Cleaned and transformed data to ensure accuracy, including data type adjustments and column filtering.
Data Modeling: Established relationships between different tables to enable cross-filter functionality.
Analytical Functions & Visualization
Dynamic Reporting: Developed interactive dashboards featuring retail and sales KPIs.
Custom Visuals: Utilized a variety of charts (Bar, Line, and Pie) to identify trends, seasonalities, and outliers.
Report Design: Applied UI/UX principles to ensure the dashboard is intuitive for stakeholders, focusing on clear navigation and logical data flow.
Cloud Collaboration
Power BI Service: Published local reports to the cloud environment.
Sharing & Workspace Management: Configured reports for sharing with trainers and peers, demonstrating a workflow suitable for corporate environments.

📊 Summary of Tasks Completed
Task 1: Built a comprehensive report from a standalone Excel workbook.
Task 2: Mastered data blending by combining local files with live OData web feeds.
Task 3: Executed advanced data loading and lab-based troubleshooting.
Task 4: Focused on high-level report design and storytelling with data.
